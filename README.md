# SalatWatch 🕋

**SalatWatch** is a premium, feature-rich Islamic companion application designed specifically for **Zepp OS** devices (including Amazfit GTR 4, GTS 4, T-Rex 2, and others). It combines modern UI aesthetics with robust utility to help users stay connected to their faith effortlessly.

---

## 🌟 Key Features

### 📅 Smart Prayer Timetable
- **GPS-Based Accuracy**: Automatically calculates precise prayer times (Fajr, Dhuhr, Asr, Maghrib, Isha) based on your current location.
- **Live Countdown**: Real-time progress bar and countdown timer to the next prayer.
- **Hijri Calendar**: Accurate Hijri date display with Islamic holiday detection.
- **Umm al-Qura & World League**: Customizable calculation methods for global accuracy.

### 🧭 Qibla Compass
- **Sensor Integration**: Uses the device's built-in magnetometer and GPS to provide a real-time direction to the Kaaba.
- **Visual Feedback**: Dynamic gold crescent indicator that highlights when you are facing the correct direction.

### 📿 Digital Tasbih
- **Haptic Counter**: High-quality vibration feedback for every tap.
- **Auto-Cycle**: Automatically cycles through *SubhanAllah*, *Alhamdulillah*, and *Allahu Akbar* after every 33 counts.
- **Set Tracking**: Keeps track of how many full sets (99 counts) you have completed.

### 💰 Zakat & Fasting Utilities
- **Zakat Calculator**: Simple, offline calculator for gold, silver, and cash assets.
- **Fasting Tracker**: Manual tracker to keep record of missed fasts and tracking progress.

### 🔔 Adhan & Notifications
- **Adhan Alarms**: Configurable background alarms with high-quality Makkah Adhan audio.
- **Ayah of the Day**: (Requires Phone Bridge) Daily Quranic verses displayed on the dashboard.

## 🛠️ Technical Structure

The project is built on the **Zepp OS 2.0/3.0 architecture**, utilizing a modular JavaScript structure for maximum performance on resource-constrained hardware.

- **`page/`**: Contains the UI logic for each distinct feature.
- **`utils/`**: Core mathematical engines for GPS prayer times and Hijri calculations.
- **`assets/`**: Optimized media assets (TGA images and MP3 audio).
- **`app-side/`**: Companion service logic for phone-to-watch communication.

## 🚀 Installation & Development

### Prerequisites
- [Zeus CLI](https://docs.zepp.com/docs/guides/tools/zeus-cli/) installed via npm.
- A Zepp OS compatible device or the [Zepp OS Simulator](https://docs.zepp.com/docs/guides/tools/simulator/).

### Commands
```bash
# Install dependencies
npm install

# Build the project
npx zeus build

# Preview on a real device (Generates QR Code)
npx zeus preview --device <device_id>

# Run in Simulator
npx zeus dev
```
